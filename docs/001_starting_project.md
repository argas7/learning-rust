# Configurando o ambiente

Basta seguir a página de [Getting Started](https://www.rust-lang.org/learn/get-started) do Rust

Comando utilizado em 2022-10-22: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`

# Primeiro projeto

## Criando projeto

Performe `cargo init` para criar um projeto default

Estrutura de arquivos e pastas criadas:
```
$root
|__src
   |__main.rs
|__target
   |__debug
   |__rustc_info.json
   |__CACHEDIR.TAG
|__.gitignore
|__Cargo.lock
|__Cargo.toml
```

## Executando pela 1ª vez

Performe `cargo run`
